# Usage

## Onboarding

Kalirova uses step-by-step onboarding to collect profile basics, unit preferences, goals, and health permissions.

## Home

The Home tab summarizes calories remaining, nutrition, weight trend, workouts, hydration, goals, and AI insights.

## Meals

Use Meals to add foods by day and meal type. AI meal estimation is optional. Before a request is sent, Kalirova shows the restaurant, item, portion, and notes that will be sent. AI estimates are never saved automatically; review, edit, and accept them first.

## Activity

Activity shows workouts, Apple Watch or HealthKit calories when available, Kalirova estimated calories, duration, distance, and heart-rate context.

## Insights

Insights uses charts and weekly summaries to show nutrition, activity, and weight trends.

## Profile

Profile contains grouped settings for personal details, units, goals, AI, privacy, HealthKit, iCloud Backup, and developer information.

## iCloud Backup

iCloud Backup is off by default. If enabled, eligible app data may sync through the user's private iCloud account. OpenAI API keys are never included in iCloud Backup.
